int oggz_purge ( OGGZ *  oggz)

Erase any input buffered in Oggz.

This discards any input read from the underlying IO system but not yet delivered as ogg_packets.

Parameters
oggzAn OGGZ handle
Return values
0Success
OGGZ_ERR_SYSTEMError seeking on underlying IO.
